3. Herkunft und Charakter der Daten

Alle angezeigten EKGs werden durch ein mathematisches Vektor-/Dipol-Modell synthetisch erzeugt. Sie basieren nicht auf realen Patientenaufzeichnungen. Die generierten Signale sind didaktisch idealisiert und können von realen EKG-Befunden in Morphologie, Variabilität, Artefakten und Komplexität erheblich abweichen. Reale EKGs sind regelmäßig mehrdeutiger, vielschichtiger und artefaktreicher als die Darstellungen in diesem Werkzeug.
